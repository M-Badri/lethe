Completely set up a mesh and its manifold for rotor-stator domains.

Parameters
[in,out]triangulation[stator] The triangulation to which a grid is attached. It will be substituted by the merged rotor-stator triangulation
[in]mesh_parameters[stator] The mesh parameters used to decide what type of mesh or primitive is used
[in]manifolds_parameters[stator] The information about the type of manifolds attached to the boundary conditions
[in]boundary_conditions[stator] The information about the boundary conditions id. This is used to set-up the periodicity of the domain
[in]mortar[rotor] The information about the mortar method control, including the rotor mesh parameters This function currently only works for dealii meshes.
